# 排序

## 题目链接

https://leetcode-cn.com/problems/sort-colors/comments/

力扣75. 颜色分类

快速排序的思考...
但實測並不會比sort快== 
何況sort只要寫一行~
    
---------------------------------------

```cpp
class Solution {
public:
    void sortColors(vector<int>& nums) {
        sort(nums.begin(), nums.end());
        int g = 0, b = nums.size()-1;
        for(int r = 0; r<nums.size(); r++){
            if(nums[r]==0){
                swap(nums[r], nums[g++]);
            }
            else if(nums[r]==2){
                swap(nums[r], nums[b--]);
            }
        }
    }
};
```
